# 베이비밀 (BabyMeal) — 서비스 설계서

## 서비스 한 줄 정의
"냉장고 재료로 우리 아이 개월수에 맞는 레시피를 뽑아주는 웹앱"

---

## 핵심 철학
- **로그인 없음** — 로컬스토리지로 아이 프로필 유지
- **3초 이내 시작** — 접속 → 바로 사용 가능
- **안전이 우선** — AI 레시피 + 개월수별 위험 재료 경고
- **가챠 재미** — 뽑기 요소로 메뉴 결정 피로 해소

---

## 페이지 구조

```
/ (메인)
├── /recipe        — 레시피 생성 (핵심)
├── /gacha         — 오늘의 레시피 랜덤 뽑기
├── /history       — 저장된 레시피 (로컬스토리지)
└── /badge         — 시도한 식재료 뱃지 컬렉션
```

## 화면별 설계

### 1. 메인 화면 (/)

```
┌─────────────────────────────────────┐
│  🍼 베이비밀                         │
│  "오늘 뭐 먹이지?" 고민 끝!           │
│                                     │
│  [아이 개월수 설정] ──── 18개월 ▾    │
│                                     │
│  ┌──────────────┐  ┌─────────────┐  │
│  │  🎰 오늘의   │  │  🥕 재료로  │  │
│  │  레시피 뽑기 │  │  만들기     │  │
│  └──────────────┘  └─────────────┘  │
│                                     │
│  최근 만든 레시피  →→               │
```

- 첫 방문: 개월수 입력 모달 → 저장
- 재방문: 이전 개월수 자동 로드

### 2. 재료로 만들기 (/recipe)

**Step 1 — 재료 선택**
```
카테고리 탭: [탄수화물] [단백질] [채소] [과일] [직접입력]

[감자] [쌀] [고구마] [단호박]  ← 태그 형태 선택
[당근] [애호박] [브로콜리] ...

선택된 재료: 감자 ✕  당근 ✕  소고기 ✕

                          [레시피 만들기 →]
```

**Step 2 — 레시피 결과**
```
┌─────────────────────────────────┐
│  🥘 소고기 감자 당근 죽          │
│  (18개월 · 후기 이유식)          │
│                                 │
│  ⚠️ 안전 체크 이상 없음          │  ← 위험 재료 없으면 초록
│  (또는 🚨 꿀 포함 — 12개월 미만 주의)│
│                                 │
│  📝 재료                         │
│  · 쌀 40g, 소고기 20g ...       │
│                                 │
│  👨‍🍳 조리법                      │
│  1. 소고기를 잘게 다진다 ...     │
│                                 │
│  💡 영양 포인트                  │
│  철분, 단백질 보충에 좋아요      │
│                                 │
│  🛒 재료 구매  [쿠팡 바로가기]   │  ← 쿠팡파트너스
│                                 │
│  [💾 저장] [🔄 다시 만들기]      │
│  (3회 이상 다시 만들기 시 → 광고)│
└─────────────────────────────────┘
```

### 3. 랜덤 뽑기 (/gacha)

```
┌─────────────────────────────────┐
│                                 │
│     오늘 뭐 먹이지? 🎰           │
│                                 │
│   개월수: 18개월                 │
│                                 │
│        [ 뽑  기 ]               │  ← 클릭 시 애니메이션
│                                 │
│  ✨ 결과: 단호박 달걀 볶음밥      │
│                                 │
│  [레시피 보기] [다시 뽑기]       │
│                                 │
└─────────────────────────────────┘
```
- 재료 선택 없이 개월수만으로 AI가 알아서 추천
- 뽑기 애니메이션 (슬롯머신 or 카드 뒤집기)
- 마음에 안 들면 다시 뽑기 (3회 이상 → 광고)

- "100가지 식재료 도전" 목표
- SNS 공유 버튼 (카카오 공유)

## 개월수별 안전 필터 (하드코딩)

| 개월수 | 주의/금지 재료 |
|--------|---------------|
| 6개월 미만 | 소금, 설탕, 꿀, 우유, 달걀, 견과류 전체 |
| 6~9개월 | 꿀, 생우유, 견과류, 가공식품, 짠 재료 |
| 9~12개월 | 꿀, 생우유, 견과류 (분말 제외) |
| 12개월 미만 공통 | 꿀 (보툴리누스 위험), 생우유 |
| 전 연령 주의 | 가시 있는 생선, 질식 위험 크기 음식 |

→ 재료 선택 시 해당 개월수 위험 재료는 빨간 경고 배지 표시
→ AI 프롬프트에도 안전 조건 포함

## 수익화 구조

| 요소 | 방법 | 시점 |
|------|------|------|

| 프리미엄 | 무제한 생성 + 주간 식단표 (월 2,900원) | 3개월 후 |

## 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 프레임워크 | Next.js 14 (App Router) | SSR + 빠른 배포 |
| 스타일 | Tailwind CSS | 빠른 UI |
| AI | Claude Haiku API | 저렴 ($0.001/회) |
| 데이터 저장 | 로컬스토리지 | 로그인 불필요 |
| 배포 | Vercel | 무료, 빠름 |
| 광고 | 카카오 애드핏 → AdSense | 단계적 |
| 제휴 | 쿠팡파트너스 | 즉시 적용 |
