# BabyMeal — Claude Code 가이드

"냉장고 재료로 우리 아이 개월수에 맞는 레시피를 뽑아주는 웹앱"
운영자: 18개월 + 40일 두 아이 아빠 개발자

---

## 핵심 원칙
- **로그인 없음** — 로컬스토리지로 아이 프로필 유지
- **안전 우선** — 개월수별 위험 재료 경고 필수 (ChatGPT 대비 차별화)
- **가챠 재미** — 랜덤 뽑기 요소
- **뱃지 수집** — 시도한 식재료 뱃지 모으기 (리텐션)

## 기술 스택
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **AI**: Claude Haiku API (`claude-haiku-4-5-20251001`)
- **Storage**: localStorage (로그인 없음)
- **Deploy**: Vercel
- **Ads**: 카카오 애드핏 → AdSense
- **Affiliate**: 쿠팡파트너스

## 페이지 구조
- `/` — 메인 (개월수 설정 + 진입점)
- `/recipe` — 재료 선택 → 레시피 생성
- `/gacha` — 랜덤 뽑기 (재료 없이 개월수만으로)
- `/history` — 저장된 레시피 목록
- `/badge` — 시도한 식재료 뱃지 컬렉션

## 상세 설계
`docs/서비스설계.md` 참고

## 수익화 (구현 우선순위)
1. 쿠팡파트너스 재료 링크 (즉시)
2. 카카오 애드핏 배너 (즉시)
3. 3회 이상 재생성 시 광고 게이트
4. AdSense 신청 (트래픽 1천+ 후)
5. 프리미엄 구독 2,900원/월 (3개월 후)

## 개월수별 위험 재료 (하드코딩 필수)
- 12개월 미만: 꿀 (보툴리누스), 생우유
- 9개월 미만: 달걀흰자 주의, 견과류 금지
- 전 연령: 소금/설탕 최소화, 가시 생선 주의

## AI 프롬프트 핵심
- 개월수 → 이유식 단계 자동 매핑 (6~8개월: 초기/미음, 9~11개월: 중기/죽, 12~15개월: 후기/진밥, 16개월~: 완료기/밥)
- 개월수 + 선택 재료 + 금지 재료 목록 → Claude Haiku 호출
- 출력: 레시피명, 재료(g), 조리법(3~5단계), 영양 포인트

## MVP 구현 순서
1. Next.js 프로젝트 + Vercel 배포
2. 로컬스토리지 아이 프로필
3. 재료 선택 UI
4. Claude API 연동
5. 위험 재료 경고
6. 쿠팡파트너스 + 광고
7. 랜덤 뽑기
8. 뱃지 시스템
